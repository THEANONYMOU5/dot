##tmux/
###Managing Panes
Create panes
    C-b -          split vertically (top/bottom)
    C-b |          split horizontally (left/right)


Switch panes (arrow keys)
       Alt-left       go to the next pane on the left
       Alt-right      (or one of these other directions)
       Alt-up
       alt-down
